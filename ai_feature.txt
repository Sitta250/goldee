Today Gold Analysis - Claude Implementation Plan

GOAL

Implement a production-safe "Today Gold Analysis" feature with:
- model: gemini-2.5-flash
- secret: GEMINI_API_KEY only
- language: Thai and English with toggle
- schedule: generate twice daily at 09:30 and 18:00 (UTC+7)
- content split:
  - Thai section = recent Thai gold price movement recap
  - Global section = major world events + expert commentary affecting gold

NON-NEGOTIABLE RULES

- Never invent prices, dates, sources, experts, or events.
- All numeric price comparisons must be computed by backend code.
- LLM summarizes only backend-curated evidence.
- LLM output must be strict JSON matching schema.
- If evidence is weak/conflicting, output conservative language and low confidence.
- No trading advice. Never use buy/sell/investment recommendation language.
- Separate observed move vs possible drivers.
- Do not claim causation unless sources clearly support it.

OUTPUT JSON SCHEMA

{
  "price_analysis": {
    "headline": "string",
    "summary": "string",
    "vs_yesterday": {
      "direction": "up|down|flat",
      "absolute_change": number,
      "percent_change": number
    },
    "vs_7d": {
      "direction": "up|down|flat",
      "absolute_change": number,
      "percent_change": number
    }
  },
  "market_drivers": [
    {
      "theme": "string",
      "impact_type": "already_affecting|could_affect",
      "summary": "string",
      "confidence": "low|medium|high",
      "source_count": number
    }
  ],
  "expert_view": {
    "overall_trend": "bullish|bearish|mixed|unclear",
    "summary": "string",
    "consensus_strength": "low|medium|high"
  },
  "disclaimer": "string"
}

PROMPT RULES FOR GEMINI

- Use computed price values exactly as provided.
- Use already_affecting only with direct/strong evidence.
- Otherwise use could_affect.
- Keep summaries concise:
  - price_analysis.summary <= 80 words
  - each market_drivers.summary <= 50 words
  - expert_view.summary <= 70 words
- Return valid JSON only.

ACCEPTANCE CRITERIA

- Works with only GEMINI_API_KEY.
- Analysis generation runs twice daily (09:30 and 18:00 UTC+7).
- Homepage shows latest valid persisted analysis.
- Thai/English toggle works across primary UI.
- Thai section focuses on local recent price movement.
- Global section focuses on major world events and expert signals.
- No fabricated values/sources/experts in output.
- Guardrails reject invalid/harmful outputs before display.

ANTI-PATTERNS TO AVOID

- Running LLM summarization on every page request.
- Asking model for reasons without evidence bundle.
- Treating random blogs/social posts as expert sources.
- Mixing Thai retail and global spot without explicit labels.
- Letting model infer causality without source support.

MINIMAL PSEUDOCODE

function buildDailyGoldAnalysis() {
  const priceFacts = computePriceFacts();
  const newsItems = rankAndDedupNews(fetchGlobalNews()).slice(0, 12);
  const expertItems = rankExperts(fetchExpertCommentary()).slice(0, 5);

  const inputBundle = { priceFacts, newsItems, expertItems };
  const llmJson = callGemini(inputBundle, "gemini-2.5-flash");

  const validated = validateOutput(llmJson, priceFacts, newsItems, expertItems);
  return persistAnalysis(validated);
}
